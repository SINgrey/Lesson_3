Git instruction

  1. git config --global user.name "[имя]" - Устанавливает имя
  2. git config --global user.email "[email]" - Установка электронной почты
  3. git init [название] - Создает новый локальный репозиторий
  4. git add [название] - Индексирует указанный файл для последующего коммита
  5. git commit -m "[сообщение с описанием]" - Фиксирует проиндексированные изменения и сохраняет их с описанием
  6. git status - Перечисляет все новые или изменённые файлы, которые нуждаются в фиксации
  7. git push [удалённый репозиторий] [ветка] - Загружает все изменения локальной ветки в удалённый репозиторий
  8. git log - История коммитов для текущей ветки
  9. git reset --hard - Сбрасывает всю историю вместе с состоянием рабочей директории до указанного коммита
  10. git reset --soft - Сбрасывает на шаг назад где можно изменить или удалить
